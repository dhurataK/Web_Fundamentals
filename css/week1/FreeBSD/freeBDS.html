FreeBDS/ ppc Project

Introduction

The FreeBSD/pcc project pages contain about the FreeBDS port to the the PowerPc® architecture. As with the port itself, these pages are still a work in progress.

Status

The FreeBSD/pcc port is still a Tier 2 platform. That means it is not being fully supported by our security officer, relase engineers and toolchain maintainers.

Latest News

  • 03 April, 2009: Support for the PowerPC G5 has been committed to SVN
  • 03 March, 2008: Support the Freescale® PowerQUICC III MPC85XX family system-on-chip development boards has been into CVS.
  • 27 February, 2008: FreeBSD 7.0 is the first release to the officially support to the FreeBSD/pcc port.
